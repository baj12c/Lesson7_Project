Why should you ensure that the script element that links to the jQuery library file comes before any other script element for a .js file that includes jQuery code?

You should ensure that the script element that links to the jQuery library file comes before any other script elements because if this element does not come first then the jQuery statements will execute before the page loads rather than after.

Question #2:  (5 points)
++++++++++++
Why should you avoid including jQuery in a project if you don't need it?


You should avoid including jQuery in a project if you don't need it because using jQuery requires that you use more files, which can complicate a project and leaves more room for possible errors.

How does every jQuery statement start?

Every jQuery statement starts with the $ symbol, which specifies that any code that follows should be interpretted using jQuery library.

What is a Content Delivery Network (CDN)?  List and explain at least FOUR advantages in using a CDN.

A Content Delivery Network represents a group of geographically dispersed servers deployed to facilitate the distribution of information generated by web publishers in a timely and efficient manner. 

Advantages: 

1.  Significantly reduces page load time, so your webpage loads quicker 
2. Provides an easier delivery of video or audio rich content
3. provides a more secure network with no geographical barriers
4. Allows for an easier management of "traffic peaks"
